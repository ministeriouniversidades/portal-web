<p><strong>&iquest;Est&aacute; cerrado el plazo para presentar una solicitud para la segunda fase del proceso de selecci&oacute;n para las ayudas de formaci&oacute;n del profesorado universitario?</strong></p>
<p>El plazo para presentar las solicitudes para la segunda fase del proceso de selecci&oacute;n para las ayudas de formaci&oacute;n del profesorado universitario se suspende durante todo el tiempo que dure el estado de alarma y se reanudar&aacute; al finalizar este.</p>
<p><strong>&iquest;Qu&eacute; pasa con mi solicitud ya presentada?</strong></p>
<p>Las solicitudes ya presentadas ser&aacute;n v&aacute;lidas y los candidatos no tendr&aacute;n la necesidad de presentarlas de nuevo.</p>
<p><strong>&iquest;Puedo presentar una solicitud para para la segunda fase del proceso de selecci&oacute;n para las ayudas de formaci&oacute;n del profesorado universitario?</strong></p>
<p>S&iacute;, puesto que el plazo est&aacute; suspendido. La aplicaci&oacute;n permitir&aacute; reanudar el proceso cuando se levante la suspensi&oacute;n.</p>
<p><strong>En mi caso, no llegu&eacute; a completar la solicitud. &iquest;Qu&eacute; pasa con la informaci&oacute;n introducida en el programa?</strong></p>
<p>Para las solicitudes que se encontraban en proceso de elaboraci&oacute;n, la aplicaci&oacute;n guardar&aacute; la informaci&oacute;n introducida hasta el momento y no ser&aacute; necesario reiniciarlas desde el principio. Deber&aacute;n completarse cuando se levante la suspensi&oacute;n.</p>
<p><strong>&iquest;C&oacute;mo puedo contactar con el Ministerio de Universidades en relaci&oacute;n con esta convocatoria?</strong></p>
<p>El buz&oacute;n de correo para solicitantes:&nbsp;<a href="mailto:solicitantes.fpu@universidades.gob.es" target="_blank" rel="noopener">solicitantes.fpu@universidades.gob.es</a>, seguir&aacute; operativo durante el estado de alarma.</p>
